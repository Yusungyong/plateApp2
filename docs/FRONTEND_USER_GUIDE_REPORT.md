# Plate 앱 프론트엔드 사용자 가이드 보고서

작성일: 2026-06-14  
작성 기준: 현재 로컬 프론트엔드 코드, 기존 문서, `play-console-assets/android`에 저장된 화면 이미지  
제한 사항: 현재 서버 미기동 상태라 실제 앱 실행 화면, 최신 API 응답, 로그인 후 데이터 상태는 검증하지 못했습니다. 아래 내용은 코드 기반 기능 인벤토리입니다.

## 1. 문서 목적

이 문서는 Plate 앱이 사용자에게 제공하는 주요 화면과 기능을 사용자 가이드 관점에서 정리한 보고서입니다. 추후 PPT로 만들 경우 각 장을 슬라이드 단위로 나누고, 기존 스크린샷 또는 서버 복구 후 새로 캡처한 이미지를 삽입하면 됩니다.

## 2. 앱 개요

Plate는 맛집과 음식 경험을 이미지, 영상, 지도, 친구 활동, 레시피로 탐색하고 기록하는 모바일 앱입니다.

사용자는 앱에서 다음 활동을 할 수 있습니다.

- 주변 맛집 콘텐츠 탐색
- 이미지 피드와 영상 피드 감상
- 맛집 검색 및 필터링
- 방문 후기 이미지 등록
- 맛집 영상 등록
- 좋아요, 댓글, 신고, 차단
- 친구 찾기, 친구 요청, 친구 활동 확인
- 레시피 탐색 및 상세 조리 정보 확인
- 프로필과 계정 정보 관리
- 알림 확인

## 3. 하단 주요 탭

앱의 하단 탭은 사용자 기준의 주요 진입점입니다.

| 탭 | 제공 화면 | 주요 목적 | 로그인 필요 |
|---|---|---|---|
| 홈 | `Home` | 콘텐츠, 주변, 제철음식 탐색 | 일부 기능만 필요 |
| 검색 | `Search` | 가게, 음식, 피드 통합 검색 | 아니오 |
| 추가 | `ImageFeedEditor` | 이미지 후기 작성 | 예 |
| 레시피 | `Recipe` | 레시피 목록과 상세 조회 | 아니오 |
| 프로필 | `MyPage` | 내 활동, 계정, 친구 관리 | 예 |

## 4. 홈 화면

홈 화면은 앱의 핵심 탐색 화면입니다. 상단 제목을 누르거나 좌우 스와이프 흐름으로 세 가지 홈 탭을 전환하는 구조입니다.

홈 내부 탭:

- 컨텐츠
- 내 주변
- 제철음식

### 4.1 컨텐츠 탭

사용자는 이미지와 영상이 섞인 홈 콘텐츠 피드를 스크롤하며 볼 수 있습니다.

주요 기능:

- 영상/이미지 콘텐츠 카드 조회
- 피드 새로고침
- 무한 스크롤로 더 보기
- 콘텐츠 클릭 시 영상 피드 또는 이미지 뷰어로 이동
- 댓글 버튼 클릭 시 해당 콘텐츠 댓글 화면 진입
- 좋아요 토글
- 검색어가 있는 경우 콘텐츠 필터링
- 데이터 없음, 로딩, 에러 상태 표시

연결 화면:

- 영상 콘텐츠: `VideoFeedScreen`
- 이미지 콘텐츠: `ImageFeedViewer`

### 4.2 내 주변 탭

사용자는 지도와 주변 콘텐츠를 기반으로 근처 맛집을 탐색할 수 있습니다.

주요 기능:

- 현재 위치 기반 주변 콘텐츠 확인
- 지도 위 맛집/피드 마커 확인
- 주변 영상과 이미지 콘텐츠 탐색
- 위치 권한이 필요한 경우 안내
- 주변 피드 새로고침
- 특정 마커 또는 콘텐츠 선택 시 상세 피드로 이동
- 전체 지도 화면으로 확장 진입

연결 화면:

- `FullScreenMap`
- `VideoFeedScreen`
- `ImageFeedViewer`

### 4.3 제철음식 탭

사용자는 월별 또는 시즌별 제철음식 정보를 탐색할 수 있습니다.

주요 기능:

- 제철음식 히어로 카드 확인
- 제철음식 항목 선택
- 제철음식 관련 개요 확인
- 관련 레시피 화면으로 이동
- 관련 지도 화면으로 이동
- 검색어 기준 제철 항목 필터링

연결 화면:

- `Recipe`
- `FullScreenMap`

## 5. 검색 화면

검색 화면은 가게, 음식, 영상, 이미지 피드를 통합 탐색하는 화면입니다.

주요 기능:

- 검색어 입력
- 검색어 지우기
- 결과 없음, 로딩, 에러 상태 확인
- 결과를 장소 단위로 그룹화해서 보기
- 2열 masonry 형태의 검색 결과 카드 확인
- 결과 더 보기
- 검색 결과 선택 시 콘텐츠 유형에 따라 이동
- 필터 모달 열기

검색 필터:

- 콘텐츠 유형
- 정렬
- 검색 반경
- 현재 위치 기반 가까운순/반경 적용
- 필터 초기화

연결 화면:

- 이미지 결과: `ImageFeedViewer`
- 영상 결과: `VideoFeedScreen`
- 장소 결과: `FullScreenMap`

## 6. 전체 지도 화면

전체 지도 화면은 장소 중심 탐색과 길찾기 보조를 담당합니다.

주요 기능:

- 전체 화면 지도 표시
- 특정 장소 중심으로 지도 포커스
- 현재 위치 중심 이동
- 목적지까지 경로 확인
- 거리와 소요 시간 요약 표시
- 지도 마커 선택 시 이미지 또는 영상 피드로 이동
- 장소명/주소 기반 후보 검색

주의 사항:

- 현재 위치, 목적지 좌표, 장소 검색 결과가 없으면 경로 안내가 제한됩니다.
- 위치 권한이 필요할 수 있습니다.

## 7. 이미지 피드

이미지 피드는 사용자의 방문 후기와 사진을 보여주는 화면입니다.

### 7.1 이미지 뷰어

주요 기능:

- 이미지 피드를 세로 스와이프로 탐색
- 한 피드 안의 여러 이미지를 확인
- 좋아요
- 좋아요 사용자 목록 보기
- 댓글 보기 및 작성
- 장소 버튼으로 지도 이동
- 작성자 또는 피드 메타 정보 확인
- 본인 게시물 수정/삭제
- 타인 게시물 신고/차단

연결 화면:

- `ImageFeedEditor`
- `FullScreenMap`

### 7.2 이미지 등록/수정

사용자는 이미지 후기 게시물을 작성하거나 수정할 수 있습니다.

주요 입력 항목:

- 방문 후기 내용
- 이미지 선택 및 미리보기
- 가게/장소 검색
- 함께 방문한 친구 입력

주요 기능:

- 이미지 라이브러리에서 사진 선택
- 이미지 리사이징 및 업로드 준비
- 이미지 형식 검증
- 기존 이미지 교체/삭제
- 장소 정보 생성 또는 연결
- 친구 방문 정보 생성
- 새 이미지 피드 등록
- 기존 이미지 피드 수정

## 8. 영상 피드

영상 피드는 짧은 맛집 영상을 세로 스와이프 형태로 소비하는 화면입니다.

### 8.1 영상 감상

주요 기능:

- 세로 페이징 영상 재생
- 영상 탭으로 일시정지/재생
- 다음 영상 자동 준비
- 썸네일/로딩 상태 표시
- 좋아요
- 좋아요 목록 보기
- 댓글 보기 및 작성
- 장소 정보 칩 클릭 시 지도 이동
- 작성자 프로필 이동
- 친구 활동 패널 확인
- 시청 진행률 저장
- 영상 완료 시청 기록 전송
- 본인 영상 수정/삭제
- 타인 영상 신고/차단

### 8.2 영상 등록/수정

사용자는 맛집 영상을 새로 등록하거나 기존 영상을 수정할 수 있습니다.

주요 입력 항목:

- 영상 파일
- 가게/장소 검색
- 함께 방문한 친구
- 영상 설명

주요 기능:

- 장소 자동완성
- 친구 자동완성
- 영상 미리보기
- 등록/수정 저장
- 편집 모드일 때 기존 값 불러오기

## 9. 레시피 화면

레시피 화면은 음식 조리 정보를 탐색하는 보조 콘텐츠 영역입니다.

### 9.1 레시피 목록

주요 기능:

- 카테고리별 레시피 조회
- 추천 레시피 가로 목록 확인
- 전체 레시피 목록 확인
- 새로고침
- 무한 스크롤
- 검색 화면으로 이동
- 알림 화면 진입

### 9.2 레시피 상세

주요 기능:

- 레시피 제목과 요약 확인
- 조리 시간, 난이도, 인분 확인
- 태그 확인
- 재료와 수량 확인
- STEP별 조리 순서 확인
- 상세 데이터 새로고침
- 로딩/에러 상태에서 다시 시도

## 10. 인증 화면

인증 영역은 로그인, 회원가입, 소셜 회원가입, 비밀번호 재설정으로 구성됩니다.

### 10.1 로그인

주요 기능:

- 이메일 또는 아이디 입력
- 비밀번호 입력
- 일반 로그인
- 회원가입 이동
- 비밀번호 찾기 이동
- Apple 로그인
- Kakao 로그인
- Google 로그인
- 소셜 로그인 결과에 따라 회원가입 추가 정보 입력으로 이동

### 10.2 회원가입

회원가입은 단계형 입력 구조입니다.

단계:

1. 약관 동의
2. 이메일 입력
3. 비밀번호 설정
4. 닉네임 입력

소셜 회원가입은 비밀번호 단계를 건너뛰고 약관, 이메일, 닉네임 흐름으로 진행됩니다.

약관 화면 기능:

- 전체 동의
- 서비스 이용약관 필수 동의
- 개인정보 처리방침 필수 동의
- 약관 문서 보기

### 10.3 비밀번호 재설정

비밀번호 재설정 흐름:

1. 이메일 입력
2. 인증 코드 입력
3. 새 비밀번호 설정

## 12. 친구 기능

친구 화면은 친구 찾기, 요청, 목록, 활동을 한 곳에서 제공합니다.

탭:

- 요청

- 받은 친구 요청 수락/거절

- 내 친구 목록 확인
- 친구 추천 가게 확인
- 친구와의 최근 방문 기록 확인

주요 기능:

- 알림 목록 확인
- 읽음 처리

## 16. 비로그인 사용자와 로그인 필요 기능

비로그인 사용자도 일부 탐색 기능은 사용할 수 있습니다.

비로그인으로 가능한 기능:

- 홈 콘텐츠 탐색
- 검색
- 레시피 목록/상세 확인
- 일부 지도 탐색
- 이미지/영상 피드 보기

로그인이 필요한 기능:

- 이미지 등록
- 프로필 진입
- 좋아요
- 댓글 작성
- 좋아요 목록 확인
- 친구 기능
- 알림 확인
- 계정 관리
- 게시물 수정/삭제

로그인이 필요한 기능을 누르면 로그인 안내 후 인증 화면으로 이동하는 흐름입니다.

## 19. 서버 복구 후 보완하면 좋은 항목

서버가 복구되면 아래 항목을 추가 확인하는 것이 좋습니다.

- 실제 로그인/회원가입 플로우 캡처
- 홈 3개 탭의 실제 데이터 표시 상태
- 검색 결과와 필터 적용 결과
- 이미지 피드 상세, 댓글, 좋아요 모달
- 영상 피드 재생, 댓글, 친구 활동 패널
- 이미지/영상 등록 성공 흐름
- 친구 요청/수락/활동 화면
- 계정 센터의 실제 사용자 데이터 표시
- 알림 클릭 시 이동 대상
- 에러/빈 상태 문구가 사용자에게 자연스러운지

## 20. 요약

현재 프론트엔드는 맛집 탐색, 이미지/영상 소비, 사용자 기록, 친구 활동, 레시피, 계정 관리까지 포함하는 소셜 맛집 앱 구조를 갖고 있습니다.

사용자 가이드 관점의 핵심 메시지는 다음과 같습니다.

- 홈에서 콘텐츠와 주변 맛집을 발견합니다.
- 검색으로 원하는 가게와 피드를 찾습니다.
- 이미지와 영상으로 방문 경험을 기록합니다.
- 좋아요와 댓글로 콘텐츠에 반응합니다.
- 친구 기능으로 지인의 방문과 추천을 확인합니다.
- 레시피와 제철음식으로 음식 탐색 범위를 넓힙니다.
- 프로필과 계정 센터에서 내 활동과 개인정보를 관리합니다.
